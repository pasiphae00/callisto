## Callisto `v0.12.2`

Small UI fixes, and macOS builds are now **Apple-notarized** — they open on first launch with no Gatekeeper prompt (no more right-click → Open / `xattr`). Highlights: copy a Safe's address from the Overview tab, monospace owner addresses in the label dialog, and roomier keystore/private-key import dialogs.

### Install

**macOS** — download `Callisto-v0.12.2-darwin-arm64.zip` (Apple silicon) or
`-darwin-amd64.zip` (Intel), unzip, and move **Callisto.app** to /Applications.
Notarized by Apple — it just opens.

**Linux** — download `Callisto-v0.12.2-linux-amd64.tar.gz` and extract.

Already running Callisto? Just use **Settings → Check for updates** — it verifies
and installs this release for you.

### Verify (optional)

```
shasum -a 256 -c SHA256SUMS
```

`SHA256SUMS` is ed25519-signed as `SHA256SUMS.sig` with the maintainer key (the
same key the in-app updater checks).
