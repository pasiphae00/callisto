## Callisto `v0.12.1`

A security-hardening release from a pre-beta internal source review. The cryptographic core (keystore, HD derivation, signed updates) was reviewed and found sound; this ships the fixes from that pass — untrusted-string sanitization, WalletConnect approval decoding, a patched image dependency, and several smaller hardening items. **Callisto has not had a formal third-party audit** — treat it as pre-audit software (test networks / small amounts).

Full findings: [docs/security-review-2026-07.md](https://codeberg.org/pasiphae/callisto/src/branch/main/docs/security-review-2026-07.md).

### Install

**macOS** — download `Callisto-v0.12.1-darwin-arm64.zip` (Apple silicon) or
`-darwin-amd64.zip` (Intel), unzip, and move **Callisto.app** to /Applications.
First launch: right-click → **Open** (once), or
`xattr -dr com.apple.quarantine /Applications/Callisto.app`.

**Linux** — download `Callisto-v0.12.1-linux-amd64.tar.gz` and extract.

Already running Callisto? Just use **Settings → Check for updates** — it verifies
and installs this release for you.

### Verify (optional)

```
shasum -a 256 -c SHA256SUMS
```

`SHA256SUMS` is ed25519-signed as `SHA256SUMS.sig` with the maintainer key (the
same key the in-app updater checks).
